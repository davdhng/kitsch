Installing Kitsch

The easiest way to install Kitsch is to simply link to the CDN.

<link rel="stylesheet" href="https://unpkg.com/kitsch-css">

You can also use it in a project with NPM.

$ npm install kitsch-css

Buttons

There are two main button styles. Default and Outlined. You can overwrite these styles if you want (but why would you?).

By default, without any styling, the buttons look like this:

<button>Default</button>



If you want an underlined button, you can add the outline class.


<button class="outline">Default</button>
